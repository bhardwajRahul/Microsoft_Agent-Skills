## ☁️ Full-Stack Azure Bundle

**Complete coverage for Azure architects and senior engineers.** Includes all skills for comprehensive Azure expertise.

This bundle includes **all skills** from the repository. Install the complete `skills/` folder for full coverage:

### Installation

```bash
# Clone and copy all skills
git clone https://github.com/MicrosoftDocs/agent-skills.git
cp -r agent-skills/skills/* ~/.copilot/skills/  # or your preferred location
```

### Included Categories

- ☁️ Compute (Functions, Container Apps, VMs, App Service)
- 🔗 Integration (Logic Apps, API Management, SignalR)
- 📊 Data & Analytics (SQL, Data Factory, Synapse, Data Explorer)
- 🤖 AI & ML (AI Services, Machine Learning, Bot Service)
- 🔒 Security & Identity (RBAC, Confidential Computing)
- 🌐 Networking (Private Link, Application Gateway)
- 🏗️ Infrastructure (ARM, Backup, Site Recovery)
- 💰 Management (Cost Management, Governance, Monitor)
- 🎮 Specialized (IoT, Digital Twins, Communication Services)

## 💡 Tips

- **Start small:** Begin with the **Essentials** bundle and add more as needed
- **Mix bundles:** You can combine skills from multiple bundles
- **Project-specific:** Install only relevant bundles for each project to keep context focused
- **Update regularly:** Run `git pull` on the repo periodically to get new skills and updates
